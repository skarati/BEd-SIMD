This repository contains the implementation of Binary Kummer Lines: BEd251 and BEd257, corresponding to the article **Binary Kummer Lines and Binary Edwards Curves: Efficient Two-way Vectorizations** published at *ACM ASIA CCS 2026*.

<h3>Authors of the codes:</h3>
<ol>
  <li><i>Sabyasachi Karati</i>,</li>
  <li><a href="https://scholar.google.com/citations?user=9etnbVwAAAAJ&hl=en" target="_blank"><i>Rittwik Hajra</i></a>, and</li>
  <li><a href="https://github.com/tan0Shri" target="_blank"><i>Tanushri Sen.</i></a></li>
</ol>

<h3>Description:</h3>
<ol>
  <li><b>BEd251-SIMD-256</b>: This directory contains implementations of two-way vectorized fixed-base and variable-base scalar multiplications using BEd251. Each subdirectory corresponds to a specific polynomial multiplication strategy, as indicated by its name.
  </li>
  <li><b>BEd257-SIMD-256</b>: This directory contains implementations of two-way vectorized fixed-base and variable-base scalar multiplications using BEd257. Each subdirectory corresponds to a specific polynomial multiplication strategy, as indicated by its name.
  </li>
</ol>

Each subdirectory includes a <code>run.sh</code> script that provides the commands required to compile and execute the corresponding code.
